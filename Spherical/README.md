# Spherical Platform

This repository contains the core libraries and tools for the Spherical platform.

> [!TIP]
> Please refer to [CONVENTIONS.md](./CONVENTIONS.md) for project structure and architectural guidelines.

## Libraries

- [**PDF Extractor**](./libs/pdf-extractor): A Go library and CLI tool to extract product specifications from PDF documents using vision-capable LLMs.
- [**Markdown Tools**](./libs/markdown-tools): A collection of Python and Shell scripts for cleaning and formatting Markdown files.

## Project Structure

```
.
├── libs/
│   ├── pdf-extractor/   # Go application
│   └── markdown-tools/  # Python/Shell scripts
├── specs/               # Feature specifications
└── ...
```
